# 🧁 Mis Notas Cute

Aplicación web para crear, organizar y gestionar notas con una estética **cute coreana** 
Desarrollada con **HTML, CSS y JavaScript** puro.

---

## Descripción

✧ Un pequeño espacio para guardar ideas, planes y recordatorios de forma linda y organizada ✧
Incluye funcionalidades interactivas y una interfaz pensada en la experiencia del usuario.

---

## 🚀 Funcionalidades

* Crear notas
* Eliminar notas con confirmación (modal personalizado)
* Editar notas directamente (inline editing)
* Asignar prioridad (alta, media, baja)
* Orden automático por prioridad
* Guardado en **localStorage**
* Guardar con tecla Enter
* Diseño **responsive**
* Interfaz estilo cute / coreano
* Preview personalizada al compartir (Open Graph)

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage
